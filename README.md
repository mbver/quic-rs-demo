# Quinn Examples: Demonstrating QUIC in Rust​
This project provides a collection of concise, focused examples demonstrating the use of the Quinn library—a Rust-native implementation of the QUIC transport protocol. 

Each example is crafted with a minimalistic, no-nonsense approach, focusing solely on essential aspects to provide clear insights into specific features without unnecessary complexity. A dedicated README.md file in each example folder offers concise instructions and explanations.​

---

## Basic
A straightforward example illustrating the setup of a QUIC server and client.​

## ALPN
Explores various Application-Layer Protocol Negotiation (ALPN) configurations to understand how TLS manages different protocol negotiation scenarios.​

## Session
Demonstrates a minimal protocol that supports authentication and session management.

## Datagram
Demonstrates QUIC datagram support in Quinn
